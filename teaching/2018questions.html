Requirements

  • regular readings

Course description

Projects on processing natural language questions. Possible areas:

  • question typology/classification
  • question generation
  • question answering
  • questions & dialog systems (chatbots)
  • questions & discourse structure

Preliminary Schedule

We will meet weekly in the first half of the course, then carry out the projects and meet in smaller groups or only as needed.

Available datasets

  • query wellformedness corpus
  • question typology - Cornell convotoolkit
  • Sarcasm Rhetorical Question corpus
  • SQuAD Stanford question answering dataset & task
  • LearningQ - educational question generation dataset
  • TREC Question Classification dataset
  • Questions Under Discussion annotated corpus (GRAINS corpus; German)
  • Amazon Question Answering dataset (product questions and answers)
  • QuAC question answering in context
  • Question similarity: Quora Question Pairs dataset
  • Information seeking vs. NIS questions: Multilingual Bible Questions dataset

dialog act datasets (selection):

  • MRDA
  • SWDA, Switchboard dialog act corpus (Chris Potts' intro)
  • iqap Indirect Questions and Answers corpus
